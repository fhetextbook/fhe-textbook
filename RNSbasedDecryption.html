D-5.5 RNS-based Decryption

This subsection will explain how to efficiently decrypt RNS-based ciphertexts for BFV, CKKS, and BGV.

D-5.5.1 BFV Decryption: 𝖣𝖾𝖼𝖱𝖭𝖲𝖡𝖥𝖵

Suppose we have a BFV ciphertext (A,B) such that B = A ⋅S + ΔM + E (where Δ = ⌊q t⌋). We decrypt the ciphertext as follows: M = ⌈B −A ⋅S Δ ⌋ (Summary D-2.3 in §D-2.3). However, RNS does not allow direct division and rounding. Therefore, we need to express this divide-and-round operation in terms of addition and multiplication.

Let’s denote 𝖼𝗍(s) = Δm + e + 𝑘𝑞 (i.e., a decryption of ciphertext 𝖼𝗍 without modulo-q reduction). In this description, we will consider only a single set of coefficients m, e, and k extracted from polynomials M, E, and K for simplicity.

As explained in §A-1.2.2, modulo arithmetic does not support direct division. Meanwhile, the special relation a b mod p = a ⋅b−1 mod p holds if b divides a and an inverse of b modulo p exists (i.e., b and p are co-prime). Inspired by this, we can express the decrypted plaintext m as follows:

m = ⌈|𝖼𝗍(s)|q Δ ⌋ = ⌊|𝖼𝗍(s)|q Δ ⌋ + er # where er ∈[0,1] is a rounding error

= ⌊|𝖼𝗍(s)|q ⋅t q⌋ + er + ed # where ed = ⌊|𝖼𝗍(s)|q Δ ⌋ − ⌊|𝖼𝗍(s)|q ⋅t q⌋ is a scaling error

= ⌊t ⋅|𝖼𝗍(s)|q q ⌋ + er + ed

= t ⋅|𝖼𝗍(s)|q −|t ⋅𝖼𝗍(s)|q q + er + ed # where |t ⋅𝖼𝗍(s)|q ≡t ⋅|𝖼𝗍(s)|q mod q, and therefore t ⋅|𝖼𝗍(s)|q −|t ⋅𝖼𝗍(s)|q is divisible by q

Now, we choose some prime number γ which is co-prime to t and q. Then, we derive the expression for γ ⋅m as follows:

γ ⋅m = γ ⋅ ⌈|𝖼𝗍(s)|q Δ ⌋

= ⌈γ ⋅|𝖼𝗍(s)|q Δ ⌋ + es′ # where es′ = γ ⋅ ⌈|𝖼𝗍(s)|q Δ ⌋ − ⌈γ ⋅|𝖼𝗍(s)|q Δ ⌋ is a multiplication error

= ⌊γ ⋅|𝖼𝗍(s)|q Δ ⌋ + es′+ er′ # where er′ ∈[0,1] is a rounding error

= ⌊γ ⋅|𝖼𝗍(s)|q ⋅t q⌋ + es′+ er′+ ed′ # where ed′ = (⌊γ ⋅|𝖼𝗍(s)|q Δ ⌋ − ⌊γ ⋅|𝖼𝗍(s)|q ⋅t q⌋) is a scaling error

= ⌊γ ⋅t ⋅|𝖼𝗍(s)|q q ⌋ + es′+ er′+ ed′

= γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q q + es′+ er′+ ed′

Next, we derive the expression for |γ ⋅m|𝛾𝑡 as follows:

|γ ⋅m|𝛾𝑡 = |γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q q + es′+ er′+ ed′|𝛾𝑡

= |γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q q |𝛾𝑡 + |es′|𝛾𝑡 + |er′|𝛾𝑡 + |ed′|𝛾𝑡

= |γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q q |𝛾𝑡 + es′+ er′+ ed′ # assuming |es′|«𝛾𝑡 2 and |er′|«𝛾𝑡 2 and |ed′|«𝛾𝑡 2

= |(γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q) ⋅q−1|𝛾𝑡 + es′+ er′+ ed′ # since γ ⋅t ⋅|𝖼𝗍(s)|q −|γ ⋅t ⋅𝖼𝗍(s)|q is divisible by q, and q is co-prime to 𝛾𝑡

= |−|γ ⋅t ⋅𝖼𝗍(s)|q ⋅q−1| 𝛾𝑡 + es′+ er′+ ed′ # since γ ⋅t ⋅|𝖼𝗍(s)|q is a multiple of 𝛾𝑡

= ||γ ⋅t ⋅𝖼𝗍(s)|q|𝛾𝑡 ⋅|−q−1|𝛾𝑡 + es′+ er′+ ed′

Given the above relation, notice that the computation result of 𝖥𝖺𝗌𝗍𝖡𝖢𝗈𝗇𝗏(γ ⋅t ⋅𝖼𝗍(s),q,𝛾𝑡) ⋅|−q−1|𝛾𝑡 can be expressed as follows:

𝖥𝖺𝗌𝗍𝖡𝖢𝗈𝗇𝗏(γ ⋅t ⋅𝖼𝗍(s),q,𝛾𝑡) ⋅|−q−1|𝛾𝑡

= ||𝛾𝑡 ⋅𝖼𝗍(s)|q + 𝑢𝑞|𝛾𝑡 ⋅|−q−1|𝛾𝑡 # where |u|≤k 2 + 1 for the base moduli q1,q2,⋯,qk

= ||𝛾𝑡 ⋅𝖼𝗍(s)|q ⋅|−q−1|𝛾𝑡 −u|𝛾𝑡

= |γ ⋅m|𝛾𝑡 −es′−er′−ed′−u # as we previously showed that |γ ⋅m|𝛾𝑡 = ||γ ⋅t ⋅𝖼𝗍(s)|q|𝛾𝑡 ⋅|−q−1|𝛾𝑡 + +es′+ er′+ ed′

= y # let’s denote the above expression as y

Then, if es′,er′,ed′,u are small enough such that |es′+ er′+ ed′+ u|< γ, then |y|γ = −es′−er′−ed′−u (as |γ ⋅m|𝛾𝑡 = 0 as a multiple of γ). Therefore, we can effectively remove the noise terms es′,er′,ed′,u and derive m as follows:

|(y −|y|γ) ⋅|γ−1|t|t

= ||γ ⋅m|𝛾𝑡 ⋅|γ−1|t|t

= ||γ ⋅m|t ⋅|γ−1|t|t # since ||γ ⋅m|𝛾𝑡|t = |γ ⋅m|t

= |m|t

, which is the final decryption of ct we wanted to compute. Let’s denote the RNS vector of y as a (yγ,yt) ∈ ℤγ × ℤt. Then, we can efficiently compute the term |(y −|y|γ) ⋅|γ−1|t|t as follows:

|(y −|y|γ) ⋅|γ−1|t|t

= |(|yγ ⋅t ⋅|t−1| γ + yt ⋅γ ⋅|γ−1| t|𝛾𝑡︷ y −|yγ ⋅t ⋅|t−1|γ + yt ⋅γ ⋅|γ−1|t|γ︷ |y|γ) ⋅|γ−1|t|t

= |(|yγ ⋅t ⋅|t−1| γ + yt ⋅γ ⋅|γ−1| t|t︷ y −|yγ ⋅t ⋅|t−1|γ + yt ⋅γ ⋅|γ−1|t|γ︷ |y|γ) ⋅|γ−1|t|t # since ||y|𝛾𝑡|t = |y|t

= |(|yγ ⋅t ⋅|t−1| γ + yt ⋅γ ⋅|γ−1| t|t︷ y −|yγ ⋅t ⋅|t−1|γ|γ︷ |y|γ) ⋅|γ−1|t|t # since yt ⋅γ ⋅|γ−1|t mod γ = 0

= |(|yt ⋅γ ⋅|γ−1| t|t︷ y −|yγ ⋅t ⋅|t−1|γ|γ︷ |y|γ) ⋅|γ−1|t|t # since yγ ⋅t ⋅|t−1|γ mod t = 0

= |(yt −yγ) ⋅|γ−1|t|t # since |γ ⋅γ−1|t = 1 and |t ⋅t−1|γ = 1

We summarize 𝖣𝖾𝖼𝖱𝖭𝖲𝖡𝖥𝖵 as follows:

⟨Summary D-5.5.1⟩ 𝖣𝖾𝖼 𝖱𝖭𝖲𝖡𝖥𝖵

Input: 𝖼𝗍(s) = Δm + e + 𝑘𝑞

1.
Pick some prime number γ which is co-prime to t and q.
2.
Compute 𝖥𝖺𝗌𝗍𝖡𝖢𝗈𝗇𝗏(|γ ⋅t ⋅𝖼𝗍(s)|q,q,𝛾𝑡) ⋅|−q−1|𝛾𝑡

= (yγ,yt) ∈ ℤγ × ℤt

3.
Compute |m|t = |(yt −yγ) ⋅|γ−1|t|t
D-5.5.2 CKKS and BGV Decryption

CKKS and BGV ciphertexts can be decrypted efficiently by performing the ModDrop operation (Summary D-5.3 in §D-5.3) to the lowest multiplicative level. After this, there remains only a single ciphertext modulus in the RNS base, so the regular decryption algorithm can be executed efficiently without any RNS components.
